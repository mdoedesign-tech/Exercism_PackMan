# Pac-Man Boolean Logic Exercise

A small Python exercise based on the classic Pac-Man arcade game.

## About

This project is part of my Python learning journey on  
[Exercism.org]

The exercise focuses on practicing basic Python boolean logic using:
- `and`
- `or`
- `not`
- boolean return values (`True` / `False`)

The functions simulate simple Pac-Man game rules such as:
- eating ghosts
- scoring points
- losing the game
- winning the game

## Functions

### `eat_ghost()`
Checks if Pac-Man can eat a ghost.

### `score()`
Checks if the player scored points.

### `lose()`
Checks if the game is lost.

### `win()`
Checks if the player has won the game.

- Python 3
